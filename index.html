III - Basic Concepts

What is it ?

How it works

Location-based vs Content-based

URL vs CID

URL = Uniform Resource Locator

CID = Content Identifier

LifeCycle

Chunks

We only have to store these chunks once.

We can download a file piecewise, checking each piece as we get it. We don't have to download the entire file first.

We can skip directly to the piece we care about. We can even chunk a file such that the chunks align with logical boundaries (e.g., video key frames).

Merkle {Tree, DAG, *}

DAG = directed acyclic graph

What is a Tree ?

What is a graph ?

A cyclic graph ?

A directed graph ?

And a DAG ?

This is also a DAG

A real Merkle Tree

Gateway

A Gateway is a computer acting as a HTTP server and a local IPFS node at the same time. A client without a local ipfs node can send a request to the gateway, which will get the file from the swarm, and then send it back to the client. 

https://{gateway URL}/ipfs/{content ID}/{optional path to resource}
